# 台式洗碗机选购规则

用于判断台式洗碗机是否值得买。

## 快速推荐

第一次买台式洗碗机，默认推荐：

| 指标 | 推荐规格 |
|---|---:|
| 容量 | 至少 5 套，优先 6 套 |
| 最大盘子尺寸 | 至少 26 cm |
| 锅具支持 | 经常做饭时，应能放 20-24 cm 小锅 |
| 水箱容量 | 至少 5 L |
| 进水方式 | 水箱 + 接水龙头双模式 |
| 洗涤温度 | 70-75 C 高温洗 |
| 烘干方式 | 热风烘干、自动开门、主动换气，至少有一种 |
| 存储换气 | 需要把碗放机内时，支持 24-72 h 换气存储 |
| 噪音 | 不高于 55 dB，越接近 50 dB 越好 |
| 单次用水 | 约 5-8 L |
| 功率 | 常见 700-1000 W |
| 前方开门空间 | 至少预留 45 cm |

## 判断规则

| 规则 | 推荐购买 | 可以考虑 | 不建议购买 |
|---|---|---|---|
| 容量 | 6 套或更大 | 5 套 | 3-4 套，除非只洗杯子和少量碗 |
| 最大盘子尺寸 | 26 cm 或更大 | 24-25 cm | 小于 24 cm |
| 洗锅能力 | 可放 20-24 cm 小锅 | 只能放很小的锅 | 经常做饭但不能洗锅 |
| 进水方式 | 水箱 + 接水龙头 | 租房且只支持水箱 | 不能接水却只支持水龙头 |
| 水箱容量 | 5 L 或更大 | 4-5 L | 小于 4 L |
| 洗涤温度 | 70 C 或更高 | 65-69 C | 主洗低于 65 C |
| 烘干方式 | 热风、自动开门或主动换气 | 余温烘干 + 换气 | 只写模糊的余温烘干 |
| 存储换气 | 24 h 或更长 | 没有存储但用户会立刻取出 | 想当碗柜用但无烘干/换气 |
| 噪音 | 55 dB 或更低 | 56-60 dB | 开放厨房且高于 60 dB |
| 单次用水 | 5-8 L | 8-10 L | 无合理原因却高于 10 L |
| 空间 | 台面放得下，门能完全打开 | 比较紧但能用 | 尺寸未知或门打不开 |

## 需求匹配

| 用户情况 | 推荐选择 |
|---|---|
| 1-2 人、轻度做饭 | 5-6 套，优先双进水 |
| 2-3 人、每天做饭 | 6 套，能洗小锅，烘干好 |
| 租房 | 水箱 + 可选接水龙头 |
| 自住房 | 接水龙头 + 固定排水 |
| 想洗完直接放机内 | 热风或自动开门 + 24-72 h 换气 |
| 小厨房 | 购买前确认宽、深、高和前方 45 cm 开门空间 |

## 避坑项

| 说法或缺失规格 | 风险 |
|---|---|
| 只强调“消毒”“母婴级” | 洗净和烘干才是核心 |
| 只写几套，不给内胆尺寸或最大盘子尺寸 | 可能放不下正常餐具 |
| 洗涤模式很多，但不写水温 | 洗净能力不清楚 |
| 烘干只写“余温烘干” | 碗和内壁可能偏湿 |
| 不写噪音 | 开放厨房风险大 |
| 售后或保修信息不清楚 | 涉水、加热电器需要稳定售后 |

## 最少追问问题

用户信息不足时，只问这些：

1. 家里几个人吃饭？
2. 是否经常做饭，需不需要洗锅？
3. 是租房还是自住房，能不能接进水管？
4. 台面尺寸和预算是多少？
