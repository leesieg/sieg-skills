# 电视选购规则

用于判断电视是否值得买。

## 快速推荐

先从真实使用需求出发：

| 使用需求 | 推荐基线 |
|---|---|
| 父母看新闻、综艺、电视台 | 4K、系统简单、投屏稳定、广告少、3 GB 内存 + 32 GB 存储 |
| 普通客厅 | 观看距离允许时优先 75 寸，MiniLED，约 1000 nit，有分区控光 |
| 明亮客厅 | MiniLED、高亮度、低反光或防眩屏 |
| 暗室看电影 | OLED 或高分区 MiniLED，高对比度，HDR 表现好 |
| PS5/Xbox/PC 游戏 | 120 Hz 或 144 Hz、HDMI 2.1、VRR、ALLM、低延迟 |
| 当电脑显示器 | 支持 4:4:4 色度、文字清晰、低延迟，锐化和降噪可关闭 |

## 按观看距离选尺寸

| 观看距离 | 推荐尺寸 |
|---|---:|
| 2.0-2.5 m | 65 寸 |
| 2.5-3.2 m | 75 寸 |
| 3.2 m 以上 | 85 寸或更大 |

## 核心规格标准

| 规则 | 推荐购买 | 可以考虑 | 不建议购买 |
|---|---|---|---|
| 分辨率 | 4K | 43 寸以下或特殊用途可用 1080p | 55 寸及以上还是 1080p |
| 刷新率 | 游戏/体育优先 120 Hz 或 144 Hz | 普通看电视 60 Hz 可用 | 原生面板不支持却宣传高刷 |
| 芯片和系统 | 主流新款电视芯片，系统流畅 | 操作不卡即可 | 老芯片，系统明显卡顿 |
| 内存和存储 | 4 GB + 64 GB 或更好 | 3 GB + 32 GB | 智能电视低于 2 GB + 16 GB |
| HDMI | 游戏/PC 需要 HDMI 2.1，最好 2 个以上 | 普通用途 HDMI 2.0 可用 | 游戏用户却没有 HDMI 2.1 |
| App 生态 | 可装常用第三方 App，投屏稳定 | 能满足用户常用软件 | 系统封闭，装不了需要的软件 |
| 视频解码 | HEVC、AV1、HDR10，电影用户优先杜比视界 | HEVC + HDR10 | 缺少用户常用片源解码 |
| 画质设置 | 可关闭运动补偿、降噪、锐化、色彩增强 | 只能部分调节 | 强制优化，不能关闭 |
| 屏幕和背光 | 按环境选择 OLED 或 MiniLED | 低预算普通 LCD 可用 | 用弱硬件宣传强 HDR |
| 低反和防眩 | 明亮客厅优先低反/防眩 | 暗光环境普通镜面可用 | 亮客厅用强反光屏 |
| 对比度 | OLED、VA 或优秀 MiniLED 分区控光 | 普通用途对比度一般也可 | 暗室电影用低对比屏 |
| 分区控光 | 75 寸 500+ 分区更稳 | 中档几十到几百个分区 | 只说 MiniLED，不写分区数 |
| 峰值亮度 | 中端 HDR 优先 1000 nit 以上 | 基础 HDR 500-1000 nit | 低于 500 nit 却宣传强 HDR |
| 色域 | DCI-P3 93% 或更高 | 普通看电视可以低一些 | 只说广色域，不给数值 |
| HDR 格式 | HDR10 + 杜比视界或 HDR10+ 更好 | 基础 HDR10 | 只写支持 HDR，但亮度很弱 |
| 运动补偿 | 有 MEMC，且可关闭 | 基础运动处理 | 强制肥皂剧感，不能关闭 |
| 声音 | 2.1 声道或独立低音更好 | 普通电视喇叭可用 | 只写影院音效，不写扬声器结构 |
| 售后 | 官方店、自营店，安装退换规则清楚 | 可靠渠道 | 大尺寸电视安装/退换规则不清楚 |

## 显示技术怎么选

| 技术 | 适合场景 | 注意事项 |
|---|---|---|
| MiniLED | 明亮客厅、混合使用、高亮 HDR | 分区数和控光水平差异很大 |
| OLED | 暗室电影、追求黑位和对比度 | 明亮客厅、反光、烧屏焦虑要考虑 |
| RGB MiniLED / SQD / 高色域方案 | 高端色彩和亮度需求 | 要同时比较真实色域、分区、亮度和价格 |
| 普通 LCD | 低预算、日常随便看 | HDR 和对比度通常较弱 |

## HDR 判断

不要只看“支持 HDR”四个字。

| HDR 因素 | 目标值 |
|---|---:|
| 峰值亮度 | 基础 500 nit，中端 1000 nit，高端 1500 nit 以上 |
| 分区控光 | 尺寸越大，分区数越要足够 |
| 色域 | 优先 DCI-P3 93%+ |
| 格式支持 | HDR10 是基础；电影用户优先杜比视界或 HDR10+ |
| 画质调节 | 必须能调节或关闭重度优化 |

## 避坑项

| 宣传词 | 风险 |
|---|---|
| “HDR 电视”，但不写亮度、分区、色域 | 可能只是能解码 HDR |
| “MiniLED”，但不写分区数 | 控光可能很弱 |
| “高刷”，但不写原生刷新率 | 可能是插帧营销 |
| “广色域”，但没有 DCI-P3 或 BT.2020 数值 | 色彩能力不可验证 |
| “AI 画质增强” | 可能过度锐化、偏色 |
| “影院级音效”，但不写扬声器结构 | 低音通常弱 |

## 最少追问问题

用户信息不足时，只问这些：

1. 观看距离和目标尺寸是多少？
2. 主要用途是看剧、电影、游戏、体育，还是当电脑显示器？
3. 房间白天亮不亮，有没有窗户反光？
4. 预算和想在哪个平台买？
